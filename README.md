# Destiny 2
Helpful scripts for Destiny 2 puzzle-solving.

## Whisper
```bash
# Execute to print solutions.
python whisper
```

## Zero Hour
```bash
# Execute with arc singe configuration.
python zero_hour arc

# Allow replacement when picking combinations.
python zero_hour void -r
```

## Requirements
 - numpy
 - pandas
 - colorama
 - argparse
